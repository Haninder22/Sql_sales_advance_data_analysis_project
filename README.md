# SQL Sales Advanced Data Analysis

This project contains my practice work on **advanced SQL data analysis**
using a sales dataset.\
The goal was to explore how SQL can be used to analyze business data and
create useful reports from it.

I used a dataset that includes **customers, products, and sales
transactions** and wrote SQL queries to generate insights and reports.

------------------------------------------------------------------------

## Files in this Project

**SQL query gold facts.sql**\
This file contains the main SQL queries I wrote to analyze the sales
data.

**gold.dim_customers.csv**\
Customer dataset containing customer information.

**gold.dim_products.csv**\
Product dataset containing product details.

**gold.fact_sales.csv**\
Sales transaction data that connects customers and products.

**gold.report_customers.csv**\
A report generated from SQL analysis showing customer related insights.

**gold.report_products.csv**\
A report generated from SQL analysis showing product performance.

------------------------------------------------------------------------
Below is an example query used in this project to analyze customer distribution and sales by age group.

SELECT 
    age_group,
    COUNT(customer_number) AS total_customers,
    SUM(total_sales) AS total_sales
FROM dbo.[gold.report_customers]
GROUP BY age_group;

##This query groups customers by age group and calculates:

Total number of customers in each group

Total sales generated by each age group

This helps understand which customer segments contribute the most to revenue.
